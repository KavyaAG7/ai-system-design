# ⚡ AI for System Design

Generate software architecture diagrams from plain English using **Groq LLaMA 3.3**, **Graphviz**, and **Gradio**.

## Demo
🚀 Live on Hugging Face: [Click Here](https://huggingface.co/spaces/agkavya/ai-system-design)

## Features
- 🤖 LLM-powered architecture generation
- 🗺️ Auto-generated color-coded diagrams
- 🛠️ Technology stack recommendations
- ⚙️ Configurable by scale, priority, team size

## Setup

### Install dependencies
apt-get install -y graphviz
pip install -r requirements.txt

### Set API key
export GROQ_API_KEY=your_key_here

### Run
python app.py

## Tech Stack
- Groq API (LLaMA 3.3 70B)
- Graphviz
- Gradio
- Python
